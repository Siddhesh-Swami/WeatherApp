# WeatherApp jhkjhkj
Simple Weather App to get the current weather of a particular city.


## Installing
Clone the repository

``` 
git clone https://github.com/sid6070/WeatherApp.git 
```

Change directory into it

``` 
cd WeatherApp
```

Install dependencies

``` 
npm install
```

Run Server

``` 
node app.js
```

In browser , Goto 
``` 
localhost:3000
```
